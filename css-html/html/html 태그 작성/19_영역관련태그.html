block 형식

            - 화면에서 공간을 세로로 분할하고 싶을 때 사용한다. 
            
            - block 형식의 태그 종류 ▼
                - h1 ~ h6 (제목)            
                - p / pre (문단) ◀ 주로 글자에 사용한다.
                - hr     (수평선)
                - div   (영역 분할) ◀ 글자, 텍스트, 그림, 각종 파일 표현에 두루 사용한다.
            
            블록 형식 태그들은 작성된 내용이 짧더라도 한 줄을 모두 차지하는 성질 때문에,
            다음 태그 내용을 한 번에 작성하지 않고, 따로 나누어 다음에 작성한다.

div 형식

            - 가장 많이 사용하는 태그이다.
            - 주로 문서를 구분지을 때 사용한다.
            - div 안에 div 를 쓰는 이중 div 형식이 많이 사용된다.
            - div 시작 태그 안에 작성한 class 는 head 부터 foot 에 이르기까지 모두 사용 가능하다.
            - p h1 pre table 등 모든 태그에 사용 가능하다.
            - class는 홈페이지를 꾸밀 때 많이 사용한다.(style 안에 사용)

inline 형식

            - 화면에서 공간을 가로로 분할하고 싶을 때 사용한다.
            - 한 공간을 모두 차지하지 않고, 내부에 작성된 내용 크기만큼만 차지한다.
            - 해당 태그에서는 width, height 와 같은 크기 지정 속성은 사용할 수 없다.
            
            - 대표적인 line 형식 태그 목록
                - b : 굵기
                - i : 기울이기
                - mark : 글 배경 색
                - span : 영역 분할

in line 형식은 가로를 세로로 분할하는 성질을 가지고 있어서, 한 줄을 전부 차지하지 않고, 작성한 만큼만 적용된다.

  • block 형식 : 가로를 모두 차지하는 영역이다.
  • inline 형식 : 내용을 가로로 작성해준 만큼만 영역을 차지한다.
