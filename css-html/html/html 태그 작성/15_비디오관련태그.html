비디오 태그

            비디오를 재생할 때 사용하는 태그
                controls : 재생, 일시정지, 음량조절 등의 기능을 사용할 수 있다.
                  source : 비디오 파일을 제공하기 위한 공간으로, 실질적으로 비디오를 불러오는 공간이다.
                autoplay : 자동 재생되게 설정할 때 사용한다.
                   muted : 비디오 음소거
                    loop : 자동으로 반복 재생되게 설정할 때 사용한다.

            사용 예제
                  video 태그 2개 내부에 source 작성
                  video 태그 자체에 controls 작성
                    video 태그 2개 내부에
                    source 태그 작성
                        source 태그 자체에 src 작성하여 비디오를 불러올 위치 지정
                        source 태그 자체에 type 작성하여 불러올 비디오 형식 지정
        
            iframe : 웹에서 또 다른 html 을 태그하거나, 비디오를 가지고 올 때 사용하는 태그이다. 
                     단, 비디오의 저작권 문제로 외부에서의 재생을 막아놨을 경우, 재생이 불가능 하다.
