iframe 사용하기

            iframe 은 웹페이지나, 웹에 있는 정보를 포함하여 보여주는 태그이다. 
            html 5 부터는 자주 사용되지 않는다.
            html 을 포함하여 보여주기도 한다.
            
            src=불러올 주소 또는 위치 작성
            width=가로 길이 지정
            height=세로 길이 지정
            frameborder=테두리를 지정할 때 사용하는 태그로, 0으로 지정하면 테두리는 생성되지 않는다.
        
            iframe 은 외부에서 태그하여 보는 시도를 차단해 둔 웹 페이지나 비디오라면
            엑스박스가 뜨면서 홈페이지에서 연결을 거부했습니다. 가 출력된다. 
            그렇지 않은 경우에는 html 에서 src 안에 작성한 홈페이지를 정상적으로 볼 수 있다.
