ol(Ordered List) : 순서가 있는 목록

            type "속성" : 표시되는 숫자 모양을 변경하고 싶을 때 사용한다.
            type "a" : 영어 소문자(a, b, c, ...) 로 시작한다.
            type "A" : 영어 대문자(A, B, C, ...) 로 시작한다.
            type "i" : 로마 소문자(i, ii, iii, iv, v, ...) 로 시작한다.
            type "I" : 로마 대문자(I, II, III, IV, V, ...) 로 시작한다. 
            type "1" : 숫자(1, 2, 3, ...) 로 시작한다.

            start "속성" : 시작 지정. 숫자만 입력 가능하다. 
               reversed : 거꾸로 시작(역순, 반대. 큰 수 ▶ 작은 수)

            작성 순서 : ol 태그 작성 후 li 태그 작성하기

dl(Description List) 태그 : 설명 목록

            dl 태그 : 설명 목록이 작성될 공간이다.(Description List)
            dt 태그 : 설명 제목(Description Title)
            dd 태그 : 설명 내용(Description Data)
