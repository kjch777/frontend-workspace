테이블 관련 추가 태그

th(Table Header) 태그

            - 테이블의 제목 또는 내용을 강조할 때 사용하는 태그이다.
            - 두꺼운 글씨, 가운데 정렬이 자동으로 적용된다.
            - 링크를 생성할 때 사용 ▶

caption 태그

            - 테이블에 대한 설명을 작성하는 태그이다.

행, 열 병합 속성 : rowspan, colspan

            rowspan : 세로로 얼마만큼 합칠 것인지 나타낼 때 사용하고, 숫자로 표기한다.
            colspan : 가로로 얼마만큼 합칠 것인지 나타낼 때 사용하고, 숫자로 표기한다.
