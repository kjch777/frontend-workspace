a(anchor : 연결, 고정하는 태그)

            
            - 현재 HTML 문서와 다른 HTML 문서를 연결하여,
              페이지를 이동할 수 있게 만드는 태그이다. 
            
            - a 태그로 감싸진 제목/내용 등을 하이퍼링크를 통해 하이퍼텍스트로 연결한다. 
                - 하이퍼링크 : Hyper(최고의) link(연결)

            - html 5 부터는 대부분의 경우 a 태그가 iframe 태그의 역할을 대신한다.
                - href="" 연결하고자 하는 웹 사이트 주소나, html 파일의 이름을 작성한다. 
                - href : HyperText REFerence. 연결할 주소를 지정하는 속성이다.
                    
                - target="_blank" 새 탭에서 열기
