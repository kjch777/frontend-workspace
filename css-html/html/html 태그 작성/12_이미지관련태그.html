이미지 태그 img

            
            - 웹 문서에 사진이나 그림 따위의 이미지를 삽입하는 용도의 태그이다. 
            
            [속성]
            (속성 : 태그에 작성하는 내용)

            1.    src : 삽입할 이미지의 경로를 작성하는 속성
                        (파일 위치, 인터넷 주소 == 이미지를 가져올 수 있는 경로)
            
            2.  width : 가로 길이를 지정해주는 속성(필수 아님)
            
            3. height : 세로 길이를 지정해주는 속성(필수 아님)
            
            4.    alt : 이미지 설명을 작성하는 속성
                        이미지가 보이지 않을 때, 화면에 출력되는 텍스트로,
                        이미지 설명을 읽어주는 역할도 같이 한다.

인터넷에 존재하는 주소로 이미지 출력하기



이미지를 절대 경로로 출력하기

            절대 경로 : 변하지 않는 기준점으로부터 목표까지의 경로를 의미한다.
                 (예 : 서울특별시 강남구 테헤란로 14길 6 남도빌딩 5층 T반)
            
            - 윈도우 기준으로는 서울 특별시가 C, D 드라이브 등의 드라이브에 해당한다.

상대 경로로 이미지 출력하기

            상대 경로 : 현재 파일/페이지 위치를 기준으로 목표까지의 경로를 나타낸 것
                       (파일 기준에서부터 시작하는 위치)
                 (예 : 학원에서 엘레베이터를 타고 1층으로 내려가서 오른쪽으로 가면 세븐일레븐에 도착한다.)

            코드에서 뒤로가기를 작성할 때는 ../ 를 이용하여 뒤로가기를 진행하면 된다. 
            cd : change directory 의 약자로, 폴더 변경을 의미한다. 
            뒤로가기를 n번 하고 싶다면 ../ 를 n번 작성해주면 된다.

고정 크기 단위(px) : 항상 고정된 크기를 지닌다.
