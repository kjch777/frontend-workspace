표 관련 기본 태그

table(표) 태그

            - 웹 문서에서 자료(데이터) 정리를 위해 사용하는 태그이다.
            - table 태그는 행, 열을 작성하기 위한 공간을 나타낸다.

table tr(Table Row) 태그

            - 테이블의 한 행을 나타내는 태그이다.

table td(Table Data) 태그

            - 한 행의 한 칼럼을 나타내는 태그이다.
              칼럼 == 엑셀에서 셀 하나
