부트스트랩 네비게이션

        부트스트랩의 네비게이션 바는 반응형 웹 디자인에서 사용되는 기능으로,
        다양한 사용자 화면 크기에 따라 알아서 적절하게 변경되는 메뉴를 쉽게 만들 수 있다. 

        [class 기능 설명]
                                     .navbar: 네비게이션 바를 정의하는 기본 설정으로, 네비게이션 바의 기본 설정 스타일을 적용
        .navbar-expand{-sm -md -lg -xl -xxl}: 네비게이션 바가 특정 화면 크기 이상에서 확장되도록 할 수 있다. 
                                              화면 크기가 설정된 값보다 작을 때에는, 햄버거 모양의 메뉴 아이콘으로 축소된다. 
                       
                       -sm: 작은 화면 크기(576px) 이하는 햄버거 모양의 메뉴 아이콘으로 보여진다. 
                       -md: 중간 화면 크기(768px) 이하는 햄버거 모양의 메뉴 아이콘으로 보여진다.(아이패드, 갤럭시 탭 등의 태블릿)
                       -lg: 큰 화면 크기(992px) 이하는 햄버거 모양의 메뉴 아이콘으로 보여진다.(작은 사이즈의 모니터 정도)
                       -xl: 큰 화면 크기(1200px) 이하는 햄버거 모양의 메뉴 아이콘으로 보여진다.(보통 사이즈의 모니터 정도)
                       -xxl: 큰 화면 크기(1400px) 이하는 햄버거 모양의 메뉴 아이콘으로 보여진다.(큰 사이즈의 모니터 정도)

        .navbar-light: 밝은 배경색을 사용하는 네비게이션 바
                       링크와 텍스트 색상은 어두워진다. 
                       
         .navbar-dark: 링크와 텍스트 색상은 밝아진다. 
                       .bg-색상 을 같이 사용해 주어야 한다. 
            .bg-light: 네비게이션 바의 배경색을 밝게 설정할 수 있다. 
             .bg-dark: 네비게이션 바의 배경색을 어둡게 설정할 수 있다. 
          .bg-primary: 네비게이션 바의 배경색을 기본 색상으로 설정할 수 있다. 

         .navbar-brand: 네비게이션 바의 브랜드나 로고를 표시하는 부분이다. 
       .navbar-toggler: 작은 화면에서 네비게이션 바를 열고 닫는 버튼이다. 
                         처럼 아무것도 없는 상태. 
                        열고 닫는 기능만 존재할 뿐, 눈에 보이는 디자인이 없는 상태이다. 

       .navbar-toggler-icon: 눈에 보이는 디자인 icon 추가. 
                             아이콘은 햄버거 모양과 비슷하다. 

              .collapse: 반응형으로 작동하는, 내용을 감싸는 부분이다. 일종의 포장과 같은 역할을 한다. 
       
       .navbar-collapse: 네비게이션 바의 내용이 확장/축소 될 수 있도록 해준다 .

       .navbar-nav: 네비게이션 바 안의 목록을 감싸는 부분이다. 
       
         .nav-item: 네비게이션 바의 각 항목을 표현해준다. 

         .nav-link: 네비게이션 바의 각 항목의 링크를 디자인할 수 있다. 

         data-bs-toggle = "collapse"
          부트스트랩에서, js를 사용하여 태그를 토글로 만들어 준다. 
         
         data-bs-target = "#id이름"
          토글 대상의 id를 나타낸다.
