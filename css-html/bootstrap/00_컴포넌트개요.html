컴포넌트 정리하기

        컴포넌트는 특정 기능이나, 기능들의 집합을 의미한다.
        독립적으로 설계된 기능이자, 재사용이 가능한 모듈 또는 단위를 의미한다. 
        (객체, 기능, 메서드 등)

Button

        클릭 가능한 버튼을 생성한다. 
        부트스트랩으로 버튼 기능 사용을 시작할 때,
        class 옆에 btn 버튼 시작 기능을 작성하고,
        추가로 적용하고 싶은 스타일을 작성한다. 
    
        input type 에 있는 버튼 기능을 사용하고 싶다면, ▼ 와 같은 형식으로 작성해 준다. 
    
        a 태그를 버튼처럼 사용하고 싶다면 ▼ 와 같은 형식으로 작성해 준다.
a 태그 버튼

버튼 테두리 설정하기

버튼 크기 조정

        버튼의 크기는 btn-lg 또는 btn-sm 으로 조정할 수도 있고,
        padding 을 활용하여 버튼의 크기를 조정할 수도 있다.

버튼 비활성화

        버튼을 비활성화 하려면 disabled 을 추가해주면 된다. 

        a 태그를 활용하여 만든 버튼을 비활성화 하려면
        disabled "aria-disabled=true" 를 작성하면 된다. 
        ▲ 앞에 disabled 도 같이 사용해 주어야 한다.
