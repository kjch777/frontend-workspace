        추가적인 스타일이 필요할 경우에는 css 를 사용하여 디자인을 만들어 주어야 한다.

라이브러리와 프레임워크의 차이

        라이브러리: 개발자가 개발을 하는 데에 있어, 사용 범위가 프레임워크보다 더 넓다. 
        프레임워크: 개발자가 개발을 하는 데에 있어, 사용 범위가 제한적이다.

class 사용 방법 정리

      container: 반응형 고정 폭을 제공한다. 
                 페이지 안의 컨텐츠를 정렬하고, 자동으로 중앙에 배치한다. 
                 기본적으로 페이지의 양 옆에 여백을 주어, 컨텐츠를 중앙에 정렬하는 것이다. 

            row: container 에서, 가로 한 줄을 나타낸다. 
                 가로 한 줄은 col-12 로 이루어져 있다. 
                 가로 한 줄의 col 이 12 를 초과할 경우,
                 자동으로 다음 줄에 배치된다. 

            col: row(가로 한 줄) 안에서, 세로로 몇 칸을 차지할 것인지 설정할 수 있다. 
                 col-1: 세로 중간사이즈(따로 표기가 없으면 자동으로 중간사이즈 설정) 1칸 차지
              col-xs-1: 세로 가장 작은 사이즈 1칸 차지
                 
                 col-4: 가로 한 줄 안에서, 4칸을 차지하게 되면,
                        가로 한 줄은 총 세로 12 칸이기 때문에, 여유 칸은 8 칸이 된다. 

                 
                 
 
                    

                    

                    

                 


                 
                 
 
                    

                    

                 


            card: 부트스트랩에 있는 카드 스타일을 적용한다. 
       card-body: 카드 본문 틀이다. 틀 안에, img title text 등을 넣어준다. 
      
      card-title: 카드 본문 안에서의 제목을 나타낸다. 
                  제목 스타일을 적용한다. 

       card-text: 카드에서, 내용을 작성하는 공간이다. 
                  텍스트 스타일을 적용한다. 

   card-img-위치: card 안에 있는 img 의 위치를 적용한다. 
                 img 가 카드 바깥으로 빠져나오지 않게 방지한다. 
                 카드 틀 안에 img 가 배치될 수 있도록 해준다. 
                 이걸 사용하면 따로 width 나 height 를 작성해 주지 않아도 된다. 

                    card-img-top: 이미지를 위에 배치한다. 
                 card-img-bottom: 이미지를 아래에 배치한다. 

      img 설정하기
        img-fluid: col 또는 특정 가로 크기에 맞게 조정해준다. 
                 col-4       img-fluid 
                 4칸 기준으로 이미지 크기가 자동으로 알맞게 들어가진다. 
                 width: 100%  height: auto 값과 같다. 
        
        img-thumbnail: 이미지에 테두리와 패딩을 추가하여, 썸네일 이미지 스타일을 적용할 수 있다. 

        rounded: 이미지의 모서리를 둥글게 만들어 준다. 
                        border-radius: 25rem
        
        rounded-circle: 이미지를 가볍게 원형으로 만들어준다. 
                        border-radius: 50%
        
        rounded-pill: 이미지의 모서리를 완전히 둥글게 만들어 준다. 
                      border-radius: 50rem

              d-flex: 태그를 flex 컨테이너로 만들어준다. 
                      태그 내부에 생기는 아이템이나 태그는 flex 에서 지정한 규칙에 따라 정렬된다. 
                      
              justify-content-start: flex 컨테이너 내부에서, 아이템이나 태그를 왼쪽 끝으로 정렬한다. 
                justify-content-end: flex 컨테이너 내부에서, 아이템이나 태그를 오른쪽 끝으로 정렬한다. 
             justify-content-center: flex 컨테이너 내부에서, 아이템이나 태그를 가운데로 정렬한다. 

             table: 테이블 태그의 시작을 알리는 기능이다.
     
     table-striped: 테이블의 홀수행과 짝수행에 서로 다른 배경색을 적용하여,
                    행 구분을 시각적으로 편하게 할 수 있도록 스타일을 적용해 놓은 것이다.
