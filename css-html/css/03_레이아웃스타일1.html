레이아웃(layout)

사용 목적을 기반으로, 시각적 효과를 구성하는 행위이다.
사전적 의미: 배치, 정리 기술적 의미: 구성 요소를 제한된 공간에 효율적으로 배치하는 것

display: 화면에 출력될 표시나 형식에 대한 배치를 지정하는 행위이다.

        태그가 화면에 어떻게 보여질 지에 대한 형식을 지정하는 속성이다.

            - block: 화면을 수직으로 분할한다.(한 줄을 전부 차지한다.)
                     + 가로(width), 세로(height) 지정 가능
            
            - inline: 화면을 수평으로 분할한다.(디자인을 글자, 문자열로 취급한다.)
                      + 가로(width), 세로(height) 지정 불가능

            - inline-block: inline처럼 수평 분할도 가능하고, block처럼 가로 세로도 지정이 가능하다. 
                            + 가로(width), 세로(height) 지정 가능

            - none: 실제로는 태그가 존재하지만, 화면에는 보이지 않게 숨겨준다. 

            - flex: 태그 사이의 간격을 유연하게 처리할 수 있도록 해주고, 
                    태그의 정렬 방향 또한 유연하게 처리하는 방식이다. 

            - grid: 격자 무늬(#) 형식으로 화면을 쉽게 분할하는 형식이다.
