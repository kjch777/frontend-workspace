정렬을 도와주는 float clear

        float: 뜨다(자바에서의 float 과는 전혀 관련 없다.)
               태그를 띄워서 좌/우로 정렬하는 속성이다. 

               float을 지정하지 않은 태그와, 지정한 태그가 겹치는 문제가 발생한다. 

        clear: float 으로 인하여 띄워져 있는 상태를 해제하는 속성이다. 
               태그 겹침 문제를 해결할 수 있다.
