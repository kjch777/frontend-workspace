태그의 여백과 관련된 스타일

        1. padding: 내용과 선 사이의 공간
                    사람이 패딩을 입었을 때 몸이 커지는 것처럼, 태그 영역이 넓어진다. 

        2. border: 테두리
                   영역이 어디까지인지 구분하기 위한 선

        3. margin: 여백, 간격
                   다른 태그와의 간격을 나타내는 영역

padding, margin, border 방향성

        000 ◀ 이 글에서 padding, margin, border 를 나타낸다. 

        000-top ◀ 위쪽 방향으로 크기를 지정한다.

        000-bottom ◀ 아래쪽 방향으로 크기를 지정한다.

        000-left ◀ 왼쪽 방향으로 크기를 지정한다.

        000-right ◀ 오른쪽 방향으로 크기를 지정한다.

        단, border(선) 속성 값은 [크기 모양 색상] 형식으로 지정해 주어야 한다.
        
        예를 들면,
            border-right: 10px dotted red
                와 같다.

padding, margin, border-width 크기 한 번에 지정하기

        padding, margin, border-width ◀ 속성 값 크기 설정
        
        속성 값 1개 ◀ 상하좌우
        padding: 20px ◀ 상하좌우로 20px 씩 크기를 주라는 뜻이다.
        margin: 20px ◀ 상하좌우로 20px 씩 크기를 주라는 뜻이다.

        속성 값 2개 ◀ 상하와 좌우로 나누어 속성 값 크기 설정
        padding: 20px 10px ◀ 상하로 20px 씩, 좌우로 10px 씩 크기를 주라는 뜻이다.
        margin: 20px 10px ◀ 상하로 20px 씩, 좌우로 10px 씩 크기를 주라는 뜻이다.

        속성 값 3개 ◀ 상, 좌우, 하로 나누어 속성 값 크기 설정
        padding: 20px 10px 30px ◀ 상 20px, 좌우로 10px 씩, 하 30px 만큼 크기를 주라는 뜻이다.
        margin: 20px 10px 30px ◀ 상 20px, 좌우로 10px 씩, 하 30px 만큼 크기를 주라는 뜻이다.

        속성 값 4개 ◀ 상, 우, 하, 좌 순서(시계방향) 로 나누어 속성 값 크기 설정
        padding: 20px 10px 30px 40px ◀ 상 20px, 우 10px, 하 30px, 좌 40px 만큼 크기를 주라는 뜻이다.
        margin: 20px 10px 30px 40px ◀ 상 20px, 우 10px, 하 30px, 좌 40px 만큼 크기를 주라는 뜻이다.

margin: 0 auto;

        다른 태그(요소) 와의 간격을 자동으로 지정하여, 가운데 정렬하게 만드는 속성 값
        
        기본적으로 자동 정렬해 준다.(= 가운데 정렬)
        
        display: block 값에서 자주 사용한다. 

        inline, inline-block 은 text-align: center; 를 이용한다.
