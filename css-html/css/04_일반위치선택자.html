일반적인 위치나 구조를 나타내는 선택자

        형제 관계에 있는 태그들 중, 특정 태그만 선택하는 선택자
        
        선택하는 기준을 통해 위치를 구분한다. 
        
        

            

        

        A ◀ 태그 선택 기준: div 태그 대신 변수명으로 작성한 A를 선택한 것이다.

        A:first-child ◀ 형제들 중, 가장 처음을 선택한 것이다. 
                         첫번째가 A가 맞다면 선택
        
        B:last-child ◀ 형제들 중, 가장 마지막을 선택한 것이다. 
                        마지막이 B가 맞다면 선택
                            

                                

                                

                                

                                

                            

                            일:nth-child(1) {
                                형제들 중, 첫번째가 일이 맞다면
                            }
                            이:nth-child(2) {
                                형제들 중, 두번째가 이가 맞다면
                            }
                            삼:nth-child(3) {
                                형제들 중, 세번째가 삼이 맞다면
                            }
                            사:nth-child(4) {
                                형제들 중, 네번째가 사가 맞다면
                            }

                만약, 태그 옆에 class나 id가 없다면, 태그명을 작성해 주어도 된다.
                    

                        

                        

                        

                    

                    .선택받은아이들 > div:nth-child(1) {
                        선택받은아이들 태그 내부에서, 가장 처음으로 작성된 div 태그를 선택한 것이다. 
                    }
                    .선택받은아이들 > div:nth-child(2) {
                        선택받은아이들 태그 내부에서, 두번째로 작성된 div 태그를 선택한 것이다. 
                    }
                    .선택받은아이들 > div:nth-child(3) {
                        선택받은아이들 태그 내부에서, 세번째로 작성된 div 태그를 선택한 것이다. 
                    }

                만약, 중간 자식들은 건너뛰고, 맨 마지막에 있는 자식만 꾸며주고 싶다면,
                    .선택받은아이들 > div:last-child {
                        선택받은아이들 태그 내부에서, 가장 마지막으로 작성된 div 태그를 선택한 것이다. 
                    }

                태그는 div span a ul li 등 모든 태그에 사용 가능하다.
