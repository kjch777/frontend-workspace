태그 위치 관련 스타일

        position 은 태그의 위치를 지정하는 속성이다. 
        
        position: relative; (상대적인)
        - 지정된 태그 내부에 다른 태그가 상대적인 위치에 지정될 수 있도록 
          기준이 될 수 있게 만드는 속성이다. 

        position: absolute; (절대적인)
        - 기본 태그의 배치 순서를 무시하고, 지정된 절대 위치에 
          태그를 배치할 수 있도록 만들어 주는 속성이다. 

        position: fixed; (고정된)
        - 항상 고정된 위치에 태그를 배치하는 속성이다. 
          화면이 움직이건 아니건 항상 같은 위치에 존재하게 만들어 준다.

        position: sticky;
        - 스크롤에서 많이 사용된다. 
          특정 스크롤 위치에 도달했을 때 고정이 되게 만들어준다. 

        - relative(상대적 위치) + fixed(고정 위치)

        - 지정된 기준에 도달하면 위치가 고정된다. 
           예) top bottom left right

        z-index 는 정수를 값으로 사용한다. 
            기본값: 0 
            숫자가 클수록 앞쪽으로 보인다. 
            숫자가 작을수록 뒷쪽으로 보인다.
