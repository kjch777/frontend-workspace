JavaScript 개념 정리

1. var, let, const 의 차이점 정리

        var: 재선언, 재할당(값변경) 전부 가능 ◀ 요즘엔 거의 사용하지 않는다. 
            ex) var 변수 = 0;
                var 변수 = 1; ◀ 재선언 가능
                    변수 = 2; ◀ 재할당 가능

        let: 재선언 불가, 재할당(값변경) 가능 ◀ 자바에서의 변수같은 존재이다. 
            ex) let 변수 = 0;
                    변수 = 1; ◀ 재할당만 가능

      const: 재선언, 재할당(값변경) 전부 불가능 ◀ 자바에서의 상수같은 존재이다. 
            ex) const 변수 = 0; ◀ 최초 선언, 최초 할당만 가능

        
        재선언과 재할당이란?
            재선언: 변수명에 대한 정의를 다시 내리는 것이다. 
            재할당: 기존에 존재하는 값에 대하여 새로운 값을 다시 저장하는 것이다.
자료형 재선언 재할당

2. 바닐라 자바스크립트(Vanilla JavaScript) 개념 정리

추가적인 라이브러리나 프레임워크 없이, 순수한 자바스크립트 코드로만 작성된 자바스크립트를 의미한다.

대표적인 라이브러리, 프레임워크: jQuery, React

예시: document.getElementById("#id이름");

3. JavaScript 객체 개념 정리

자바스크립트에서의 객체란, 데이터를 구조화하여 저장하는 데에 사용하는 기본 단위이다.

키(Key) - 값(Value) 쌍으로 이루어져 있다.

객체는 중괄호 { } 를 사용하여 정의한다.

키(Key) 는 문자 또는 문자열로만, 값(Value) 은 어떠한 데이터 타입도 전부 가능하다.

예시: 

 const 사람객체 = {
                        "이름" : "홍길동",
                        '나이' : 30,
                        "인사말" : function() {
                            alert('환영합니다.');
                        }
                    } 
                    여기서 사람 객체는 이름과 나이라는 속성을 가지고 있는 것이며,
                    인사말이라는 메서드를 가지고 있는 것이다. 

                    사람객체
                                속성: 이름, 나이
                        메서드(기능): 인사말

DOM(Document Object Model) 개념 정리

        DOM(Document Object Model) 의 약자로,
        HTML 문서의 구조화된 표현을 나타낸다. 
        HTML 문서의 각 태그를 객체로 표현한 것이다. 

        
        ▲ 버튼 태그를 객체로 표현해 본다면? ▼
        const button = {
            type: "button",
            class: 'btn',
            클릭버튼(): function() {
                alert("버튼이 클릭되었습니다.");
            }
        }

        button 객체
                    속성: type, class
            메서드(기능): 클릭버튼()

백엔드 / 프론트엔드 / 풀스택 개발자의 차이점 정리

        백엔드 개발자
            주로 서버, 데이터베이스, 데이터베이스와 프론트엔드의 연결고리를 맡는 직군이다. 
            
            서버 언어 (java, JavaScript(node.js), python 등...)
            
            데이터베이스 언어 (SQL(oracle, mysql 등...))
                주소 값을 통한 개발 및 서버 관리, 데이터 저장 및 처리 관리 등

        
        프론트엔드 개발자
            HTML, CSS, JavaScript 등을 사용하여, 웹 페이지를 구성하고 디자인 하는 직군이다. 
            쿠키, 세션, 캐시 등을 관리하기도 한다. 

            프레임워크 (BootStrap, React, jQuery, Vue 등...) 를 사용하여 개발한다. 

        
        풀스택 개발자
            프론트엔드와 백엔드 개발 모두를 수행하는 직군이다. 
            전체적인 구조를 이해해야 하고, 클라이언트와 서버 양쪽 모두 작업할 수 있어야 한다.
