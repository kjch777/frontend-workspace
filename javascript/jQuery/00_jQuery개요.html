자바스크립트 언어를 간편하게 사용할 수 있도록 단순화시킨
        오픈 소스 기반의 자바스크립트 라이브러리이다. 

        사용 방법
            1. 다운로드: 제이쿼리 홈페이지에서 다운로드
                제이쿼리 다운로드 하기
                
                

            2. CDN(주소참조활용): Content Delivery Network, 네트워크(인터넷) 에서 컨텐트를 가져오겠다는 뜻이다. 
                
                

        [기본 문법]
            1. id 문법
                $(document).ready(function() { // 문서에서, 어떠한 행동을 할 준비가 되어있다는 의미의 코드이다. 
                    $("#id이름").click(function() { // id이름이 지정된 태그를 클릭했을 때 실행할 동작
                        동작에 관련된 코드 작성
                    })
                })

            2. 이벤트리스너 문법 (onclick, keyon, onchange, ...)
                function 이벤트리스너에지정한이름() {
                    $("id이름").기능작성();
                }

            3. 서로 다른 id값 중, 원하는 tag 를 선택할 때 사용하는 문법
                $("#id이름 tag이름") ◀ 띄어쓰기를 작성해 주면 된다. 

        [선택자 문법]
            1. .css() 선택 사용
                $("태그 이름 또는 #id이름 또는 .class이름").css("배경", "색상");

            2. .toggle() 선택 사용
                $("태그 이름 또는 #id이름 또는 .class이름").toggle("속도조절"); // "속도조절" 은 생략 가능하다. 

            3. .slideToggle() 선택 사용: 토글과 비슷하지만, 주로 이미지를 보여주거나 숨길 때 사용한다. 

            4. .hide() .show() 선택 사용

            5. .append() 선택 사용: 태그나 값을 추가할 때 사용한다. 

            6. .val() 선택 사용: input, textarea, select 등 폼 내부에 있는 태그의 값을 가지고 오거나 설정하는 데에 사용한다. 
                                값을 가져올 때는 $("tag이름 또는 #id이름 또는 .class이름").val(); 형식으
                                    ▶("input").val();
                                    ▶("#아이디이름").val();
                                    ▶(".클래스이름").val();

                                값을 설정할 때는 $("tag이름 또는 #id이름 또는 .class이름").val("새로운 값 설정"); 형식으
                                    ▶("input").val("새로운 값 설정");
                                    ▶("#아이디이름").val("새로운 값 설정");
                                    ▶(".클래스이름").val("새로운 값 설정");
                                    
            7. this: 현재 실행 중인 함수(태그) 의 문맥을 나타낸다. 
                     this 이벤트가 발생한 태그를 참조한다. 
                     $("#inputField").on("input", function() {
                        console.log(this); // 이벤트가 발생한 input 태그를 가리킨다. 
                     })

            8. .on() 선택 사용: 이벤트 기능으로, 하나 이상의 이벤트를 처리한다. 
                               $("tag이름 또는 #id이름 또는 .class이름").on("event", function() {

                               })
                               event 에 속하는 유형: click, input, mouseover, mouseenter, ...
                               event 이름은 바꾸지 말아야 한다. 

                               on 같은 경우에는, on 에 해당하는 이벤트가 많기 때문에, on 을 작성한 다음
                               on("이벤트이름", function() { }) 형식으로 이벤트 이름을 작성해 주어야 하지만,
                               다른 속성들은 "이벤트이름" 을 작성해 주지 않아도 된다. 
                
            9. .keyup() 선택 사용: 사용자가 입력 창(input, textarea, ...) 에 키를 입력할 때마다 발생하는 이벤트를 감지한다. 
                                  키보드의 키가 눌릴 때마다 발생한다. 

            10. .each() 선택 사용: for 문 처럼, 지정한 태그, 클래스이름, 아이디이름을 반복하는 기능을 한다. 
                                  $("li").each(function() {

                                  })
                                  ▲ li 태그에 해당하는 모든 값을 가져와서, 반복해서 수행하고자 하는 기능을 진행한다. 

            11. .indexOf() 선택 사용: 문자열 또는 문자에서, 첫번째로 발생한 인덱스를 반환한다. 
                                     찾지 못할 경우, -1 을 반환한다.(에러 발생 최소화)

                                           index =     0       1
                                     const 과일들 = ["사과", "바나나"];
                                     const 바나나찾기 = 과일들.indexOf("바나나");
                                     console.log(바나나찾기); ◀ 1 이 출력된다. 

                                     const 수박찾기 = 과일들.indexOf("수박");
                                     console.log(바나나찾기); ◀ -1 이 출력된다.(에러 발생 최소화)

                                     .indexOf() 를 사용했을 때, 검색했던 키워드를 지우고 나면 문자열은 "" 이 되는게 맞지만,

                                     keyup 이벤트에서, 검색어가 빈 문자열이라면 모든 항목을 보여주도록 설정되어 있다. 

                                     .indexOf() 에서, "" 은 모든 값에 잠재적으로 저장되어 있다. 
                                     따라서, "" 은 index 0 의 값을 가지고 있다.
