요소 접근 방법

DOM(Document Object Model)

        웹 문서(html) 의 모든 태그를 객체 형식으로 표현한 것이다. 
        (html 태그(body, hN, pre, span 등) 를 javaScript 에서는 객체라고 표현한다.)
        
        html 태그 = javaScript 에서는 모두 객체이다. 
        ex) body 태그 = body 객체

        ▶ html 문서 안에서, 특정 태그에 접근하는 것이 가능하다. 
           특정 태그에 접근하여, 속성 값을 새로 만들거나 변경하거나 얻어올 수 있다. 
           
           document.getElementById("id이름값");
           어떤 태그인지는 알 수 없지만, 'id이름값' 으로 지정한 태그 정보를 가져온다는 의미이다.

DOM 을 이용하여 태그에 접근하기

        1. id 로 접근하기: document.getElementById("id이름값");
            ▶ id 이름이 일치하는 태그를 가져온다. 

        2. class 로 접근하기: document.getElementByClassName("class이름값");
            ▶ class 이름이 일치하는 태그를 가져온다. 
               class 이름이 일치하는 태그가 많을 경우(1개가 아닐 경우) ,
               모두 가져와서 배열 형태로 저장한 다음 전달한다. 

        3. name 으로 접근하기: document.getElementByName("name이름값");
           ▶ name 이 일치하는 모든 요소를 가져와서 배열 형태로 저장한 다음, 전달한다.

태그 속성 정리

      DOM 을 이용하여 태그에 접근하고, 접근한 태그에 추가하거나, 수정하거나, 제거하는 역할을 해 주는 속성이 있다. 

      textContent: 태그에서, 자식 태그의 모든 텍스트 내용을 보여준다. 
                   HTML 태그를 무시하고, 텍스트만 가져오거나 설정할 때 사용한다. 
                   또한, 외부 공격을 방지할 때 사용한다. 
                   innerText 와 같지만 다른 방식으로 작동한다. 
                   innerText 와 쓰임은 똑같다. 

      innerText: 태그에서, 자식 태그의 모든 텍스트 내용을 보여준다. 
                 HTML 태그를 무시하고, 텍스트만 가져오거나 설정할 때 사용한다. 
                 css 를 새로 설정해서 적용하거나,
                 추가적인 스타일 변경이 필요할 경우에 사용한다. 
                 외부 공격 방지는 하지 못한다. 

      value: input 이나 textarea 등의 값을 가져오거나, 설정할 때 사용한다. 
             이외에도 checkbox, radio, button 등, 다양한 폼 태그 값을 다룰 때 유용하다. 
             태그 안에 사용자가 입력하거나, 개발자가 입력해 놓은 값을 가지고 오는 것이다. 

      innerText: 태그 안에 작성된 텍스트를 선택하여, 변경하거나 추가하거나 삭제하는 등
             필요한 작업을 진행할 수 있다. 
  
      innerHTML: html 안에 html 을 작성한다. 보안에 취약하다. 

      style: html 태그의 스타일을 설정하거나 가져올 때 사용한다. 
             스타일을 유동적으로 변경할 수 있다. 
             색상, 글꼴, 배경색 등을 설정할 수 있다. 
             보이기, 숨기기도 가능하다.  

      className: html 태그에서, 클래스 이름을 가져오거나 설정할 때 사용한다.
