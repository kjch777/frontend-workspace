배열 Array

        - 변수를 묶음으로 다루는 것이다. 
          ▶ 자바스크립트에서는, 자료형과 상관없이 여러 값을 한 번에 다룬다. 

        - JavaScript 배열 특징
            1. 자료형 제한 X(String int 등 구분 X, 하지만 보통 한 가지 자료형만 저장한다.)
            2. 길이 제한 X(길이가 자동으로 늘었다, 줄었다 한다.)

        [배열 선언 방법]
        1. const arr = new Array(); // 0 칸 짜리 비어있는 배열 생성
        2. const arr = []; // 0 칸 짜리 비어있는 배열 생성
        3. const arr = new Array(3); // 3 칸 짜리 배열을 생성하되, 안에 있는 값은 비어있는 상태이다. 
        4. const arr = [값1, 값2, 값3]; // 3칸 짜리 배열을 생성하고, 각 칸의 값이 채워져 있는 상태이다.
