DOM(Document Object Model)

        HTML(웹 문서, 웹 페이지) 문서를 객체 기반으로 표현한 것이다. 
        ▶ HTML 문서에 작성된 내용을 트리 구조(계층형으로 표현) 로 나타냈을 때,
           각 태그, text, comment 등을 node 라고 한다. 

        document: {
            DOCTYPE: html
            HTML: {
                HEAD: {
                    TITLE: {TEXT: "문서 제목"},
                    STYLE: {CSS 내용},
                    META: {charset: "utf-8", ...},
                },
                BODY: {
                    H1: {TEXT: "제목"},
                    DIV: {...}
                }
            }
        }
