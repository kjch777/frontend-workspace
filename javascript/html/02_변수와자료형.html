변수(Variable)

        * 값을 계속 바꿔가면서 메모리에 저장할 수 있는 공간
          변하는 수
        
        * 변수에 이름을 붙이고(변수명) , 원하는 값을 넣어주면
          변수명이 필요한 곳에서 변수명을 호출하여 지정된 값을 사용할 수 있다. 

        
변수 이름(변수명) 작명 규칙

        - 의미있는 이름으로 작성할 것(a, b, c 이런 형식으로 작성하는 것은, 나중에 이게 무슨 값인지 알기 힘드므로 좋지 않다.)
        
        - 특수 문자는 _(언더 스코어) 만 사용 가능하다. (단, 맨 앞 글자에 쓸 수는 없다.)

        - 대/소문자 구분지어 작성할 것(중간에 대문자를 작성해 주는 카멜 표기법으로 작성하는 것이 좋다.)
        
        [카멜 표기법(Camel Case)]
        문장에서 여러 단어를 이어 하나의 변수명으로 작성할 때, 이어지는 첫번째 단어를 대문자로 작성하는 표기법이다. 
        membername ▶ memberName

        변수명 명명 규칙
        특수 문자 중, $ 는 혼자 사용할 수도 있고,
        $숫자 형식으로 사용할 수도 있고,
        $변수명 형식으로 맨 앞에 사용할 수도 있다.
        
        하지만, 추후 jQuery 표기 문법으로 사용 할 문자이기 때문에, 웬만하면 변수명으로는 쓰지 않는 것이 좋다. 

        $ 는 어떠한 표현식을 표현할 때 사용하기도 한다.

Java Script 변수의 종류와 특징

          var: 변수, 변수명 중복 가능(덮어쓰기 가능)
            ▶ 변수명 재사용 가능, 변수명 안에 있는 값 변경 가능
          
          let: 변수, 변수명 중복 불가능
            ▶ 변수명 재사용 불가능, 변수명 안에 있는 값 변경 가능

        const: 상수, 변수명 중복 불가능(가장 많이 쓰인다.)
            ▶ 모두 불가능

자바스크립트의 자료형

        - 문자열 String 을 표현할 때는
            const 문자열 = "hello";

        - 정수 int 를 표현할 때는
            const 정수 = 1;

        - 논리 값을 표현할 때는(boolean true false)
            const 논리값 = true; // 진실

        - null ▶ 참조하는 값이 없을 때 표시된다. *주로 오타에서 발생한다.
