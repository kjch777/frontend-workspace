조건문

        프로그래밍 코드가 해석되는 순서를 제어하는 제어문의 한 종류로,
        작성된 조건식에 따라 수행되는 코드를 변경하는 구문이다.

if(만약) 문

        
            if (조건식) {
                조건식이 true 일 경우 실행 할 코드;
            }
        
        *조건식: 결과가 true, false 인 식으로만 작성해 주어야 한다.

if - else 문

        
            if (조건식) {
                조건식이 true 일 경우 수행할 코드 작성;
            } else {
                조건식이 false 일 경우 수행할 코드 작성;
            }
        
        *else 는 조건식을 작성하는 ( ) 가 없다.
0 ~ 100 사이의 랜덤한 숫자를 발생시켜, 홀/짝 판별하기

if(만약) - else if(아니라면 만약) - else(모두 아니라면)

      
        if(조건식1) {
          조건식1 이 true 인 경우 실행할 코드 작성;
        } else if(조건식2) {
          조건식1 이 false 이고, 조건식2 가 true 일 경우 실행할 코드 작성;
        } else {
          조건식1, 조건식2 가 모두 false 일 경우 실행할 코드 작성;
        }
어린이, 청소년, 성인 구분하기
      나이를 입력 받아 
      0 ~ 13세: 어린이
      14 ~ 19세: 청소년
      20 ~ n세: 성인
