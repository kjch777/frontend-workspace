for 문

        - 프로그램의 흐름을 제어하는 제어문 중 하나로,
          특정 코드를 반복할 때 사용하는 구문이다. 

        - 끝이 정해져 있는 반복에서 주로 사용한다. 
          (예를 들어, 몇 번만 반복하고 끝낼 때)

        [작성법]
            for(1.초기식; 2.조건식; 3.증감식) {
                4.조건식이 true인 경우 반복해서 수행 할 코드 작성;
            }

            초기식: for 문을 시작할 초기 값 설정
                ex) 0 부터 시작하거나, 큰 수부터 거꾸로 시작할 수 있다. 

            조건식: for 문의 반복 여부를 지정하는 형식
                - true 일 때 반복, false 일 때 반복 종료

            증감식: 초기식에 사용된 값을 변환 시켜, 2.조건식으로 넘기는 식

            이중 for 문이나 for-each 문도 사용 가능하다.

while 문

        - 끝이 정확하게 정해져 있지 않은 반복에서 주로 사용한다. 

        [작성법]
        while(true 인 조건식) {
            조건식이 true 일 경우, 반복 수행할 코드 작성;
        }

        - false 가 나올 때까지 무한 반복한다.

분기문

        - 반복문 수행 중, 반복을 멈추거나 넘어가는 구문이다. 

        - break: (가장 가까운 / 감싸고 있는) 반복문을 멈춘다. 

        - contine: (가장 가까운 / 감싸고 있는) 반복문의 시작으로 이동한다. 
                   ▶ 다음 반복으로 넘어간다.
