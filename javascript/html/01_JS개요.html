Script 언어란?

        기본 프로그램의 동작을 사용자의 요구에 맞게 수행되도록 만들어 주는 용도의 언어이다. 
        JAVA 와는 달리, 코드를 컴파일(이진수로 번역) 하지 않고,
        Interpreter(인터프리터. 한 줄씩 코드를 읽어서 해석하는 기능) 를 수행하여 코드를 바로 실행한다. 
        
            - 장점: 컴파일에 시간을 소모할 필요가 없다. 
            - 단점: 프로그램 실행 속도가 자바에 비해 상대적으로 느리다.

javascript 코드 작성 방법

  1. 브라우저 개발자 도구(F12) ▶ console 에 직접 작성한다. * 콘솔이란, 웹 화면에서 직접 보면서 코드를 작성할 수 있는 공간이다.
  2. HTML 내부에 script 태그를 이용하여 작성해보기
