함수(function)

        - 원하는 결과를 만들기 위한 코드들의 묶음이다. 

        - 코드들을 { } 블럭으로 묶어 작성하고, 붙여진 이름을 호출하여 사용한다. 

        function 함수이름() {
            // 함수 정의
        }

        함수이름(); // 함수 호출

매개 변수(Parameter), 전달 인자(Arguments)

        매개 변수(Parameter): 중간에 개입하여 값을 변환한다. 
            - 함수 선언 시에 ( ) 안에 작성되는 변수이다. 
                ▶ 전달 인자로부터 받은 값을 저장한다. 
                ▶ 함수 정의 부분에서 사용 가능하다. 

        전달 인자(Arguments)
            - 함수 호출 시에 ( ) 안에 작성되는 값이다. 
                ▶ 함수 매개 변수로 전달되는 값이다.
